# 🛒 Product Page — Интернет-магазин (Frontend)

Интерактивная страница интернет-магазина с корзиной, фильтрацией и пользовательской логикой.
Проект реализован на **HTML, CSS и JavaScript без библиотек**.

---

## 🚀 Возможности

✅ Просмотр списка товаров
✅ Добавление товаров в корзину
✅ Увеличение / уменьшение количества
✅ Удаление товаров из корзины
✅ Подсчёт общей суммы 💰
✅ Фильтрация по категории
✅ Фильтрация по цене
✅ Поиск товаров 🔍
✅ LocalStorage (сохранение корзины)
✅ Баланс пользователя 💳
✅ Проверка на недостаток средств
✅ Toast-уведомления 🔔
✅ Loader (эффект загрузки) ⏳
✅ Панель корзины (sidebar UI)
✅ Overlay затемнение
✅ Logout система

## 🎯 Как пользоваться

* 🔍 Вводи текст в поиск — фильтрация по названию
* 📂 Выбирай категорию
* 💲 Фильтруй по цене
* 🛒 Нажми "Купить" — товар добавится в корзину
* ➕ / ➖ — изменить количество
* ✖ — удалить товар
* 🧹 "Очистить корзину" — удалить всё
* 🛍 Клик по иконке корзины — открыть панель
* 🚪 Logout — выйти из аккаунта

---

## 💾 Хранение данных

* Корзина сохраняется в **localStorage**
* Пользователь (`currentUser`) также хранится в localStorage
* При перезагрузке данные не теряются

---

## 💰 Баланс

* Начальный баланс: **5000$**
* При покупке деньги списываются
* Если средств недостаточно → показывается ошибка ❌
