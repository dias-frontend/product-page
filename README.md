# 🛒 Product Page — Интернет-магазин (Frontend)

Интерактивная страница интернет-магазина с корзиной, фильтрацией и пользовательской логикой.  
Проект реализован на HTML, CSS и JavaScript.

---

## 🚀 Демо

👉 https://dias-frontend.github.io/product-page/

---

## 📌 Функционал

- Просмотр товаров  
- Добавление в корзину  
- Изменение количества  
- Удаление товаров  
- Подсчёт суммы  
- Поиск товаров  
- Фильтрация по категории  
- Фильтрация по цене  
- Сохранение данных (LocalStorage)  
- Баланс пользователя  
- Проверка на недостаток средств  
- Toast уведомления  
- Loader эффект  
- Панель корзины  
- Logout  

---

## 🛠 Технологии

HTML5  
CSS3 (Flexbox, Grid, Animations)  
JavaScript (DOM, Events, LocalStorage)

## 🎯 Как пользоваться

- Введи текст в поиск  
- Выбери категорию  
- Фильтруй по цене  
- Нажми "Купить"  
- Используй + / -  
- Удаляй товары  
- Очисти корзину  
- Открой корзину  
- Logout  

---

## 🧠 Как работает

- Данные хранятся в массиве  
- Используется localStorage  
- Интерфейс обновляется через render()
